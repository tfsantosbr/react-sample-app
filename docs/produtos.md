# Produtos

## Tela de Cadastro

| Campo                 | Tipo                      |
|-----------------------|---------------------------|
| ID                    | Somente Leitura           |
| Nome                  | Text                      |
| Descrição             | Textarea                  |
| Categoria             | Select                    |
| Preço                 | Numérico com Decimal      |
| Imagens               | Arquivo (Aba separada)    |
| Data de Fabricação    | Data                      |
| Ativo                 | Checkbox                  |

- Cadastrar
- Cancelar

## Tela Listagem

| Campo                 | Tipo                      |
|-----------------------|---------------------------|
| Nome                  | Text                      |
| Categoria             | Text                      |
| Preço                 | Decimal com formatação    |
| Ativo                 | Sim ou Não                |
| Editar                | Link                      |
| Remover               | Ação                      |

- Filtro por Nome
- Filtro por Categoria
- Filtro por Ativos
- Paginação
- Ordenação
